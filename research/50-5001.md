# 通道 (50, 5001) 分析 — **沪市 基金/可转债 L2 聚合快照 (M201, MD201 Stream)**

## 1) 流量画像

- 帧数: **2498**
- 总字节: **1,575,767**
- 平均帧长: ~631 byte
- 压缩: ~51.6% ZIP (1290/2498)
- 40B 外层头: `flag_28=0x021faf72`, `flag_36=0`

## 3) body 结构

- ZIP 解压后是 **"M201" 聚合快照** — 与 47/4701、49/4901 同协议
- 核心差异:
  - `category` = `0x00000C03` (独有)
  - `MDStreamID` = **"MD201"**
  - `SecurityID` = 6 位 + 2 空格 (和 47/4701 一样)
  - 带**中文 GBK 简称** (和 47/4701 一样, 与 49/4901 的空白 Symbol 不同)
  - `TradingPhaseCode` = **"T111    "** (和股票一样, 不是 "T 01")
  - `NoMDEntries` = 14 (5 档 Bid + 5 档 Ask + Last + Open + High + Low)

## 9) 结论

**(50, 5001) = 沪市 可转债 / 基金 / REITs 的 L2 聚合快照 (M201 协议 MD201 stream)**。

结构和 47/4701 几乎同构, 只是换了 MDStreamID + category, 客户端用同一个 `SecurityEntryView` + `MDEntryView` 就能解析。

**非逐笔通道**。
