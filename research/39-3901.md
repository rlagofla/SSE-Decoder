## 4) 记录结构 — 与 9/5803 同构

```
PMAP (1B)
TemplateID (2B = 1e bd, FAST=3901)
SecurityID (6B ASCII)       —— PMAP 可省略 (继承前一条)
Action (1B char)            —— 'T' / 'A' / 'D' / 'C', PMAP 可省略
[随 Action 不同的字段组]
BSFlag (1B char)            —— 'B' / 'S' (FAST int = 0xC2 / 0xD3)

```

## 8) ApplSeqNum 递增序列

连续抽取 20 条记录的 4B ApplSeqNum (第 4 字段):

```
03 51 54 df → 03 51 54 e0 → 03 51 54 e1 → 03 51 54 e2 → 03 51 54 e3 →
03 51 54 e5 → 03 51 54 e6 → 03 51 54 e7 → 03 51 54 e8 → 03 51 54 e9 →
03 51 54 ec → 03 51 54 ed → 03 51 54 ef → 03 51 54 f0 → ...
```

- 'A'/'T' 事件严格 +1
- 偶有跳 1~2: 对应 'D'/'C' 事件的 4B 字段是**被引用的旧 OrderApplSeqNum**, 与本条 ApplSeq 无关; 而本条的 ApplSeqNum 被 PMAP 省略 (继承 delta)

## 11) 结论

**(39, 3901) = 沪市 债券/可转债/回购/REITs 逐笔行情 混合通道**:

- ✅ **逐笔成交 Trade** — 3,990 条 (6.4%)
