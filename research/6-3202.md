# 通道 (6, 3202) 分析 — **沪市 L2 全量快照 (股票, FAST 编码)**

## 1) 流量画像

- 帧数: **5750** (仅次于 47/4701)
- 总字节: **14,610,217** (第 2 大)
- 平均帧长: ~2.5 KB
- 压缩: **ZIP** (zip_ok=4459 / 5750 = 77.5%, 有 2 次 zip_fail — body 偶尔非 ZIP 明文)
- 40B 外层头: `compressed=1`, `flag_28=0x08000000`, `flag_36=1`

## 3) body 字节特征

- ZIP + raw-deflate, 解压后约 4~5 KB
- **FAST stop-bit 编码**, 无 M 标签, 无 FIX
- 每帧包含**多支证券的完整 L2 快照**, 每支证券块约 300~500 byte
- 显著 ASCII 标记 (FAST 字符串字段):
  - `54 52 41 44 c5` = **"TRADE"** (5 字节 FAST ASCII — 市场分段标识)
  - `54 31 31 b1` = **"T111"** (4 字节 FAST, 交易阶段码 = 连续竞价)

## 5) 各段解读

| 段 | 字段          | 推断含义                                     |
|----|---------------|----------------------------------------------|
| A  | 行首标记 12B  | frame 内行首 / time marker / template_id     |
| B  | 5 价格        | 开 / 高 / 低 / 前收 / 最新  (×10⁻²?)          |
| C  | "TRADE"+"T111"| MarketSegment (TRAD = 连续交易) + Phase Code |
| D  | 8~10 字段     | 成交量 / 笔数 / 成交金额 / 更新时间 / 均价 / 振幅 |
| E  | 10 档 Bid     | 买 1..买 10 的价 + 量 + 可能档位编号         |
| F  | 近 10 笔成交  | 最近 N 笔 逐笔成交 delta (price, qty, ΔT)    |
| G  | 10 档 Ask     | 卖 1..卖 10                                  |
| H  | 再一组        | 可能是另一时间窗口的成交历史 / 委托队列快照  |

**关键观察**: 段 F/H 的 `b8 ...` 序列嵌入了**最近成交的成交明细**, 但这**不是 逐笔通道** — 只是快照内附带的"成交队列回溯窗口", 用于帮助客户端补齐最近 N 条 trade 状态。

主板 A + ETF 混合 —— 覆盖面与 47/4701 类似, 但**表达格式完全不同**:

| 通道     | 编码       | 布局                    | 10 档盘口 | 近 N 笔成交 |
|----------|-----------|--------------------------|-----------|-------------|
| 47/4701  | M201 BE定长| 固定 69+19×N             | 5 档      | 无           |
| 6/3202   | FAST+ZIP  | 变长 FAST stop-bit       | **10 档** | **嵌入**     |

6/3202 是 **"更完整的 L2 快照变体"** —— 带 10 档 + 最近成交, 用 FAST 压缩后体积仍然 <M201 的 2 倍。

## 7) 为什么不是逐笔

- **每支证券都带 "T111" 阶段码** — 逐笔消息不带阶段码
- **每支证券都带 10 档盘口** — 逐笔消息不带盘口
- **每支证券都带聚合统计** (成交量, 笔数, 成交额) — 逐笔仅报单笔增量
- **帧频率 vs 逐笔**: 5750 帧 < 9/5803 的 ... 等 wait 实际 3560 帧. 但单帧聚合多支证券, 总信息量大

## 8) 结论

**(6, 3202) = 沪市股票/ETF 的 L2 完整快照 (FAST+ZIP 变体)**。

与 47/4701 (M201 定长快照) 的关系是"同样数据的两种编码":
- 47/4701: 盛立 ODP 格式的 M201 聚合快照, BE 定长, 5 档盘口, 无近期成交
- 6/3202: FAST 编码的扩展快照, 10 档盘口, 含最近成交历史窗口

客户端可二选一订阅, 或同时订阅做互校验。**非逐笔通道**, 不是本次寻找目标。
